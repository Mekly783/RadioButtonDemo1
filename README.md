# RadioButtonDemo1
2. Description
A Java Swing application built in Apache NetBeans.  
It uses radio buttons to select a pet (Bird, Cat, Dog, Rabbit, Pig).  
When a pet is selected, the app shows a message box and displays the corresponding image.
## Features
- Five radio buttons grouped together (Bird, Cat, Dog, Rabbit, Pig).
- Only one radio button can be selected at a time.
- Shows a message box with the selected pet name.
- Displays the pet image in a JLabel.
## How to Run
1. Clone or download this repository.
2. Open the project in Apache NetBeans.
3. Make sure the `images` folder is inside `src` with files:
   - bird.jpg
   - cat.jpg
   - dog.jpg
   - rabbit.jpg
   - pig.jpg
4. Run `RadioButtonDemo1.java`.
